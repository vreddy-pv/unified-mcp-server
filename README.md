# Unified MCP Server

This project contains a unified Model Context Protocol (MCP) server that can interact with multiple backend services.

## Functionality

The unified MCP server is designed to be a central hub for interacting with various services. It can be configured to route requests to the appropriate service based on the message content.

## Getting Started

To run the server, execute the following command:

```bash
python server.py
```
